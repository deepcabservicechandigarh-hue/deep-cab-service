# Deep Cab Service 🚖

Welcome to the **Deep Cab Service** project! 

Yeh ek cab booking aur taxi service website ka front-end code hai, jo khaskar Chandigarh aur uske aaspaas ke areas mein safar ko aasan banane ke liye design kiya jaa raha hai.

## 🛠️ Technologies Used
* HTML5

## 🚀 How to Run
Is project ko check karne ke liye bas repository ki `index.html` file ko apne kisi bhi web browser (jaise Google Chrome ya Safari) mein open karein.
